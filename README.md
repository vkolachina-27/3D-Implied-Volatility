# 3D-Implied-Volatility
A Python app that visualizes SPY options' implied volatility across various parameters (risk-free rate, dividend yield, strike price, etc.) using real-time data from Yahoo Finance, and deployed on Streamlit. It calculates implied volatility via the Black-Scholes model and plots a 3D surface showing variations by time to expiration and strike price.

Link to visualization: https://impliedvol.streamlit.app/
